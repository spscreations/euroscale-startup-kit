# EuroScale — Project Context

EuroScale is a European PlanetScale alternative: serverless MySQL DBaaS running on EU cloud infrastructure (Hetzner). Customers get a connection string — nothing else to manage.

## Architecture
- **Infra:** 3 × Hetzner CAX21 nodes (2 Nuremberg + 1 Helsinki) — K3s v1.34
- **Database:** Vitess v24 (multi-region cluster, 1 primary + 2 replicas)
- **API:** Go gRPC service (CreateDatabase, DeleteDatabase, ListDatabases, RotateCredentials)
- **Dashboard:** Next.js 16.2, React 19.2.7, Tailwind v4, TypeScript 6.0
- **Backups:** 5-layer — live replication, Vitess S3 (MinIO), rclone off-site, etcd, weekly restore test
- **Monitoring:** kube-prometheus-stack + Grafana
- **CI/CD:** GitHub Actions (build → GHCR → kubectl deploy)
- **Cost:** ~€36/mo (3 × CAX21 @ €10.49 + Storage Box)
